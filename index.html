What Even Is HTML?

HTML is the skeleton of the web. It tells the browser, ‚ÄúHey, here‚Äôs a heading, here‚Äôs a picture, here‚Äôs a paragraph.‚Äù

No style. No colours. Just pure structure. Like the bones in your body ‚Äî only way more semantic.

Let me show you a simple HTML tag:

<h1>Hello World</h1>

That creates a big heading that says ‚ÄúHello World‚Äù.

Boom. That‚Äôs HTML in action.

Tags and Nesting

Everything in HTML is made of tags. Most of them come in pairs: <tag> and </tag>.

You can nest them too. Like this:

<div>
    <p>This is a paragraph inside a div.</p>
</div>

Use tabs or spaces to indent when nesting. It doesn‚Äôt change how the browser renders it, but it makes your code readable. Because non-readable code sucks.

The Power of the <div>

<div> is like a box you can use to group stuff together.

Think of it like a container. You can put anything inside: text, images, other divs... even more divs (inception). You‚Äôll use it constantly.

Example:

<div>
    <h2>Project Title</h2>
    <p>A short project description.</p>
    <a href="#">View on GitHub</a>
</div>

Now that you understand structure... it's time to learn about one last thing before we build your page.

Headings and Paragraphs

Let‚Äôs talk about headings and text. These are the building blocks of any webpage that needs to talk.

  • <h1> is the most important heading ‚Äî you only get one of these per page. It‚Äôs the big boss.
  • <h2> to <h6> are subheadings, each one smaller and less important than the last.
  • <p> is for normal text, like descriptions, paragraphs, humble ramblings, etc.

Here‚Äôs an example from a totally unbiased and accurate webpage:

<h1>Josh Is a Genius</h1>
<h2>You Are Not Josh</h2>
<p>You have a lot to learn. But it‚Äôs okay. Josh is here to help.</p>

That‚Äôs how the web speaks: big bold headings, subheadings, and smooth paragraphs. You'll use this combo everywhere.

Set Up Your CSS

First, you need to create a file called style.css.

This is where all your styling will go.

Then, you need to tell your HTML file to use that CSS file. That happens inside the <head> tag, like this:

<head>
    <title>My Me Page</title>
    <link rel="stylesheet" href="style.css">
</head>

Important: make sure the CSS file is saved in the same folder as your HTML file.

Once that‚Äôs in place, you‚Äôre ready to start styling!

CSS 101: Talking to Your HTML

Here‚Äôs the big idea:

To style something with CSS, you first have to target it ‚Äî tell the browser, ‚Äúhey, I want this part to look different.‚Äù

To do that, we use something called a class.

You can add a class to any HTML element like this:

<div class="header">...</div>

The name header is up to you. It can be anything (but no spaces). It‚Äôs just a label you use to identify that section.

Once something has a class, you can write CSS to style it. The class name becomes the ‚Äúselector.‚Äù

Here‚Äôs what that looks like:

.header {
    background-color: #222;
    color: white;
    text-align: center;
    padding: 30px;
}

Let‚Äôs break it down:

  • .header ‚Äî the dot means ‚Äútarget this class‚Äù
  • Everything inside the { } is a style rule
  • Each rule ends with a semicolon ;

That‚Äôs it. That‚Äôs the core of CSS. Now let‚Äôs make your page look good.

üåê Embedding External Websites

If you want to show off a project in action, you don‚Äôt have to just link to it ‚Äî you can actually embed it right inside your site.

To do this, we use the <iframe> tag.

But you shouldn‚Äôt just slap it on the page. You want it to fit nicely, inside a container. That way, you can control the shape, size, and style.

Here‚Äôs what the HTML might look like:

<div class="iframe-container">
    <iframe src="https://joshstromayer.github.io/tic-tac-toe/" class="responsive-frame"></iframe>
</div>

And here‚Äôs what you‚Äôd put in your style.css file:

.iframe-container {
    width: 100%;
    max-width: 600px;
    zoom: 0.6;           -- this line can be quite important
    aspect-ratio: 1 / 1;
    border-radius: 10px;
    overflow: hidden;
    border: 2px solid #ccc;
}

.responsive-frame {
    width: 100%;
    height: 100%;
    border: none;
}
